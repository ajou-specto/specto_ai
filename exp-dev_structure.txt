# 개발환경 구조 (ChatGPT3 in Spring Boot environment)

1. User -> Controller
- 사용자는 API로 prompt를 입력하여 API Call 수행

2. Controller -> Service
- Controller에서 해당 end point를 받아주고 Service 호출

3. Service -> Servicelmpl -> ChatGPT3
- Servicelmpl에서는 RestTemplate를 이용하여
  ChatGPT3로 SecretKey와 함께 주요 정보 전달

4. ChatGPT3 -> Servicelmpl -> Controller -> User
- 처리된 응답 결과 반환
